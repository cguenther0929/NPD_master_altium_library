# Altium Master Library 

__DESCRIPTION:__
This repository is home to the master library files that Altium projects shall point to. The library files contained herein shall supersede other library files, as this library will be kept up-to-date.  

__FILE STRUCTURE__

</pre>

## Dependencies
All library file types as needed by Altium exist in this directory.
